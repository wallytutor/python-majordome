# Thermodynamics solver with automatic differentiation

**WORK IN PROGRESS:** this is not ready for use from Python and currently is not integrated to core Majordome. It is a standalone project meant to be exposed in the near future as a Majordome module.

## Status

- Parameterization of data in MaierKelley, NASA7, NASA9, Shomate, and GibbsPolynomial.

- Lua-based dynamic database system for species definition.
